# leon_customs

Vehicle tuning script for FiveM. Clean, simple and works with ESX.

![Screenshot](https://cdn.discordapp.com/attachments/1146364137119690792/1522176391297236992/image.png?ex=6a4784bc&is=6a46333c&hm=397d7449601b4b8e87c6e39154f456a779c55c98a5fe2be46676eea1bb2df909&)

## What it does

- Full vehicle tuning (performance, visuals, colors, neon, etc.)
- Preview mods before buying
- Rotate vehicle in tuning menu using arrow keys
- Smooth camera transitions
- Clean UI without any bloat

## Requirements

- ESX (es_extended)

## Installation

1. Download this repo
2. Put the folder in your `resources/` directory
3. Rename it to `leon_customs` if you haven't already
4. Add `ensure leon_customs` to your `server.cfg`
5. Edit the config to set up tuning location and prices

## Configuration

Open `config.lua` and change whatever you want:

- Tuning location (default is at the customs in Los Santos)
- Interaction distance and key
- Prices for all mods
- Camera settings
- And more

## How to use

1. Get in your vehicle
2. Drive to the tuning location (check config for coords)
3. Press E to open the menu
4. Select mods you want
5. Click "Install Upgrades" to buy them

That's it.

## Notes

- Make sure you have enough money before installing
- You can cancel anytime to revert changes
- Neon, xenon, turbo and other stuff all works
- Wheel types are supported too
